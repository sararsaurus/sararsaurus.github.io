learnings

  • Carefully creating sessions and tokens when building your app is key to preventing Cross-Site Request Forgery (CSRF) attacks. Input sanitization, particularly the removal of necessary coding characters and dangerous keywords, is important to defending against Cross-Site Scripting (XSS).

  • Getting at the key of a hash is tricky, but Ruby's each method helpfully offers an easy way to access it in a loop!
    •             
              items = { "chair" => 100, "book" => 14 }
              hash = {}
              items.each do |name, price|
              hash[name] = price
              end
              p hash
              # {"chair"=>100, "book"=>14}
  • Best practices for PTT radio use: No Hollywood lingo! Listen first before transmitting, rehearse what you're going to say, depress the talk switch for 1-2 seconds before talking, and release the button when you're done.

  • Found a news API! I bet that's what most "good news" apps are using. I also signed up to be a beta tester for a new tech news aggregator and I'll be curious to see whether it uses a similar API.
